About Style

My favorite part about styling is the CSS one. First, I started choosing a color palette. I imagined one based on violet color, because I love that color and it would look amazing and elegant with just variations and white. All my images are made by AI for the backgrounds, used Dall-E and Photoshop to add my designs and make it look more elegant and sophisticated.
My logos and some images (The footer's Cartoon, for example) are made in Adobe Illustrator, but everything else is made in Adobe Photoshop because I like it better. Every photo (except for the profile pic on the "About Me" Section) is organic, just took it and edited in Adobe Lightroom, but the pic on the "About Me" Section is an edit with Ai that I just made to look more fancy. When I had all the material, I started to make my mockup on Figma. Honestly, I didn't use it a lot, because at the beginning this was going to be just a tiny page to show some of my illustrations, but when I figured out how big my project was becoming, I started to make tiny sketches and save some prototyping time. That's why not all of my project is on Figma, because I didn't make everything there. Used some Jitter tools (one of my favorite Figma's plugins) to animate some stuff and design the main page's skills squares.

Nerdy Stuff

As i said before, this was just a small project at the beggining, so started using a lot of GIF files for the animations, because i didn't planned to use a lot of javascript, but when the webpage became bigger it started to become slower because of the GIFs, so i had to replicate a lot of the animations with JS. There's a library that saved my life for this project, i didn't know about it but GPT4 recommended it to me. It's called "Animations on Screen (AOS), it's an Angular based library wich allowed me to animate my fade in and fade out scroll animations just by importing the library and adding every id to my project." For animations as this terminal, or a lot of interactive stuff i used pure CSS because that's the challenge about this page, if i wanna show you my skills i gotta use less to be more hahaha. The gravity effect in some pics as the main cartoon pic or the certification's section is made with vanilla JS, a function that allows me to detect the mouse and add a "gravity effect" moving on 0.25 the image over X & Y coordinates. Also, added an "Observer", wich allows me to charge the gifs only when you're watching them, i loved it! BTW, all those videos are maded with Jitter. For the Whatsapp Chat, it's just an API that i found on the internet, it's called "Elfsight" and it helped me a lot. For the Caroussels used OWL Caroussel, wich is a JavaScript librery for simple and fast caroussels. Also got some CSS animations from universe.io and a lot of code automation with Copilot and GPT4. My backend it's kinda tricky. I connected the only one input field on the "Contact" Section to PHP and insert that data into PhpmyAdmin using XAMPP. Honestly i don't really like it, but it works and was something relatively easy. All my files are storaged on WIX hahah, why? because WIX UI is nice and cheap, also while i was working on ZOEK i learned to use it very well, so every < scr > is connected to wix. But please, this is not a Wix built page, i made it all by my own. You can check all my code for free on my GitHub! please check it out :)
